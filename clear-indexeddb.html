🗑️ Clear IndexedDB Data

⚠️ Important: This will permanently delete all data stored in IndexedDB for this domain. This is necessary to ensure the application uses the new PostgreSQL backend instead of old browser storage.

What is IndexedDB?

IndexedDB is browser storage that the old version of the application used to store data locally. The new version uses a PostgreSQL database on the server instead.

Why clear it?

If you're seeing unexpected data in the UI that doesn't match what's in PostgreSQL, it's likely because the browser still has old IndexedDB data that's being displayed.

Actions

Manual Steps (Alternative)

You can also clear IndexedDB manually using browser DevTools:

  1. Press F12 to open DevTools
  2. Go to the Application tab (Chrome) or Storage tab (Firefox)
  3. In the left sidebar, expand IndexedDB
  4. Right-click on CompetitionAccountDB and select Delete database
  5. Refresh the page (Ctrl+Shift+R or Cmd+Shift+R)

After Clearing

  1. Close this page
  2. Go back to http://localhost:8080
  3. Hard refresh the page: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  4. The application should now show only data from PostgreSQL
